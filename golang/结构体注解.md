## JSON

### 设置时间的格式和时区
```go
type A struct {
    Birthday time.Time `form:"birthday" time_format:"2006-01-02" time_utc:"1"`
}
```

### uint 对应JSON的string，
数据库中的主键使用`int64`或者`uint64`。在json格式返回数据时，使用int类型返回主键会到导致数据溢出（高32位的数据丢失）。因为JSON标准中int只有32位。解决办法是将64位的组件转换为string类型。
```go
type A struct {
	id uint `json:"id,string"`
}
```
